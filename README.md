# Event Management Backend

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Dayjs

## Features

- Profile Management
- Event Management
- Multi-timezone Support
- Event Update History
- Cursor Pagination

## Installation

```bash
npm install
```

Copy

```
.env.example
```

to

```
.env
```

Run

```bash
npm run dev
```
